# 浮士德 · Faust

个人主页，写代码，做项目，记录技术上的思考与实践。

## 技术栈

| 类别   | 技术           |
|------|--------------|
| 前端框架 | Vue 3        |
| 语言   | TypeScript   |
| 样式   | Tailwind CSS |
| 构建工具 | Vite         |

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 部署

```bash
npm run deploy
```

部署至 GitHub Pages，访问地址：`https://github.com/www12354jkl/faust.git`
